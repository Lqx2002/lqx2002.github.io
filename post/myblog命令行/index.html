Myblog命令行

Dec 19, 2022
1 min read
  • 安装hugo 1.wget https://github.com/gohugoio/hugo/releases/download/v0.80.0/hugo_0.80.0_Linux-64bit.deb
    2.dpkg -i hugo_0.80.0_Linux-64bit.deb
  • 生成博客
    1.hugo new site myblog
    2.git clone https://github.com/vaga/hugo-theme-m10c.git m10c
    3.hugo server -t m10c –buildDrafts
  • 编写文章
    1.hugo new post/linux.md
    2.hugo new post/socket_code.md
    3.hugo –theme=m10c –baseUrl=“https://lqx2002.github.io” –buildDrafts
  • 上传github
    1.git init
    2.git add *
    3.git commit -m “第一次提交”
    4.git remote add origin https://github.com/Lqx2002/lqx2002.github.io.git
    5.git config –global credential.helper store
    6.git push -u origin master
